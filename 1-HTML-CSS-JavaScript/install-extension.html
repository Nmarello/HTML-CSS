To install and use VS Code's Live Server Extension

Install | Use

Istallation Instructions

  1. In Visual Studio Code click the extensions icon inthe left tool bar
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use live server.
  • In VS Code's Explorer, right click (windows) or control +click (Mac) on an HTML file. From the context menu that appears, select open with live server.
